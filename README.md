# 🧠 Backpropagation with Artificial Neural Networks (ANN)

A hands-on practice project implementing **Backpropagation** in Artificial Neural Networks using Deep Learning techniques — covering both **Classification** and **Regression** tasks.

---

## 📌 Overview

This project explores the fundamentals of how neural networks learn through the backpropagation algorithm. By working through two separate notebooks, the project demonstrates how ANNs can be applied to different types of supervised learning problems — predicting discrete classes (classification) and continuous values (regression).

---

## 📁 Project Structure

```
Backpropagation-Deep-Learning-ANN-/
│
├── backpropagation-classification DL .ipynb   # ANN for Classification tasks
├── backpropagation-regression DL.ipynb        # ANN for Regression tasks
└── README.md
```

---

## 🔍 Notebooks

### 1. `backpropagation-classification DL .ipynb`
Implements a deep neural network to solve a **classification problem** using backpropagation. Covers:
- Data preprocessing & encoding
- Building and compiling the ANN model
- Forward pass and loss computation
- Backpropagation and weight updates
- Model evaluation with accuracy metrics

### 2. `backpropagation-regression DL.ipynb`
Implements a deep neural network to solve a **regression problem** using backpropagation. Covers:
- Data preprocessing & feature scaling
- Building and compiling the ANN model
- Forward pass and loss computation
- Backpropagation and weight updates
- Model evaluation with regression metrics (e.g., MSE, MAE)

## 🚀 Getting Started

### Prerequisites
Ensure you have Python 3.7+ and the following packages installed:

```bash
pip install tensorflow numpy pandas scikit-learn matplotlib seaborn
```

### Run the Notebooks

1. Clone the repository:
   ```bash
   git clone https://github.com/sanzidd/Backpropagation-Deep-Learning-ANN-.git
   cd Backpropagation-Deep-Learning-ANN-
   ```

2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Open either notebook and run the cells sequentially.

## 🧩 Concepts Covered

- **Artificial Neural Networks (ANN)** — Architecture, layers, neurons, and activation functions
- **Forward Propagation** — Computing predictions through the network
- **Loss Functions** — Binary/Categorical Cross-Entropy (classification), MSE (regression)
- **Backpropagation** — Computing gradients via the chain rule
- **Gradient Descent & Optimizers** — Weight update mechanisms (e.g., Adam, SGD)
- **Overfitting Prevention** — Dropout, regularization techniques
